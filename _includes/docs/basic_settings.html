Option that applies to wordpress post types only:

Choosing the display position

Choose to display the my movie database view to appear before or after the content area of your post.

  • after => 'After content'
  • before => 'Before content'

The default value is 'after'.

Options that apply to wordpress post types and shortcodes:

Choosing the template to use

Current templates available with the plugin are the tabs and the accordion .

The template file options are :

  • tabs => 'With tabs'
  • accordion => 'With accordion'
  • custom => 'Custom template'
  • The template files are located inside the "mmdb_templates" folder of the 'my_movie_database" plugin.

The default value is 'With tabs' which uses the "tabs.php" file located inside the "mmdb_templates" folder of the 'my_movie_database" plugin.

If you want to make your own template use the custom.php file that is found in the same folder and select 'Custom template'. The custom template is empty by default

If you want to edit the default templates or use the custom template, please copy the "mmdb_templates" folder inside your active theme folder. This way any changes you make to your template files will not be overwritten by a plugin update.

Choosing the screen width

Depending on the width setup/style of your target page, you can select from a predetermined set of width combinations for a best fit. This setting only affects bootstrap multiple column arrangements (for now) as seen on sections like cast and crew.

For example, if you have a full-width area with a no sidebar layout you would choose ' large ', ' medium ' if you have one sidebar and ' small ' for a two sidebar area. The default value is ‘medium’.

The width options and the bootstrap classes rendered for multiple column arrangements:

large => for Full-width => renders 'col-lg-3 col-md-3 col-sm-6'
medium => for one sidebar =>renders 'col-lg-3 col-md-4 col-sm-6'
small => ‘for two sidebars =>renders 'col-lg-4 col-md-6 col-sm-6'

Hide sections

Select the sections to be hidden.

The option names are different for each ressource type (movie, tvshow, person) but the options always refer to either a complete tab / accordion section, or the overview text on the main (1st) section.

  • The overview text found at the bottom of the first tab => 'Overview Text'
  • The second tab / section => ' Cast' (or 'Movie Roles' for persons)
  • The third tab /section => 'Crew' (or 'Tv Roles / Appearances' for persons)
  • The fourth tab / section => 'Trailer', 'Seasons' or 'Crew Credits'
If there is no content available from TMDb for a given section, that section will be hidden by default. For example, if your movie has no trailer or your person has no crew credits than those tabs / sections will be hidden automatically regardless of this setting.

Header Background Color

The background color for the template 'header' area

The template 'header' area will affect your tabs or accordion headers. In the case of tabs, the selected and the hovered tabs will get this color as well.

This setting in combination with the 'Background color for the template body' gives 2 color schemes to your tabs and accordion templates.

The default value is '#265a88'

Body Color

The background color for the template 'body' area

The default value is '#DCDCDC'
